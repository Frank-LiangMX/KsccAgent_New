# GenericAgent 对齐任务清单

最后更新：2026-05-06 (Phase 2+3+4+5 完成)

## 目标说明

在不推翻当前 `KsccUI`（PyQt 桌面产品形态）的前提下，对齐 GenericAgent 的核心能力：

- 更强的任务执行循环（计划 -> 执行 -> 反思 -> 修正）
- 更强的分层记忆与任务技能沉淀
- 更强的可观测性与自动化能力（同时保持可控和安全）

## 完全实现 GenericAgent 功能的任务拆分（Roadmap）

> 说明：以下按“收益 / 风险 / 实施顺序”拆分，建议逐阶段验收，不做一次性大改。

## Phase 1：执行内核对齐（高优先）

目标：让 Agent 从“单轮对话驱动”升级为“任务状态机驱动”。

- [x] P1-1 定义统一 TaskState 结构（plan/steps/results/reflection/errors）
- [x] P1-2 实现执行循环状态机（Plan -> Execute -> Reflect -> Next）
- [x] P1-3 增加失败恢复策略（重试/降级/改写子任务）
- [x] P1-4 增加每步运行日志（可追踪每一步输入输出）
- [x] P1-5 UI 增加任务步骤视图（最小版）

验收标准：

- 同一复杂任务可自动分步完成
- 出错后可自动恢复至少一次
- 用户可看到清晰的步骤轨迹

## Phase 2：Skill 晶化自动化（高优先）

目标：从“人工保存 skill 草稿”升级为“半自动 -> 自动沉淀闭环”。

- [x] P2-1 Skill 草稿质量评分（完整性、复用性、成功信号）
- [x] P2-2 自动候选入库策略（阈值 + 人工兜底）
- [x] P2-3 Skill 退化与淘汰机制（低命中/低收益）
- [x] P2-4 Skill 冲突仲裁（相似技能聚合）
- [x] P2-5 增加 Skill 效果统计面板

验收标准：

- Skill 自动增长且无明显污染
- 命中收益可量化（耗时下降、成功率提升）

## Phase 3：分层记忆增强（高优先）

目标：补齐 GenericAgent 风格的层次化记忆检索与注入调度。

- [x] P3-1 新增 Insight Index 层（跨任务可检索摘要）
- [x] P3-2 任务类型识别 -> 选择性注入策略
- [x] P3-3 记忆压缩策略（短中长期不同规则）
- [x] P3-4 记忆冲突检测（过期事实/矛盾事实）
- [x] P3-5 记忆命中可视化（本轮命中来源）

验收标准：

- 长会话 token 成本可控
- 相关记忆命中率明显提升

## Phase 4：工具能力与安全策略（中高优先）

目标：对齐示例场景能力，同时确保桌面产品安全可控。

- [x] P4-1 抽象 Tool Capability 层（`tool_executor.py` 注册表 + 风险分级 + 安全路径校验）
- [x] P4-2 高风险动作审批策略（Shell 黑名单 + 多级风险确认 + CRITICAL 自动拦截）
- [x] P4-3 网页自动化与页面证据采集（`evidence/web/` 自动保存抓取结果）
- [x] P4-4 设备自动化（ADB）适配与隔离（`adb_command` 工具 + 危险命令过滤 + `evidence/device/` 日志）
- [x] P4-5 敏感任务风控模板（支付/删除/配置/部署 4 类模板 + 关键词匹配 + UI 警告）

验收标准：

- 自动化能力可用且可解释
- 高风险操作默认可拦截

## Phase 5：评估与产品化（中优先）

目标：让”能力提升”可被度量、回归、长期维护。

- [x] P5-1 建立标准任务基准集（15 个任务，覆盖 file_ops/coding/shell/task_mode/memory/safety/documentation 7 类）
- [x] P5-2 自动回归脚本（`run_benchmark.py`，支持按类别/难度筛选，`--compare` 对比两次运行）
- [x] P5-3 指标看板（`metrics_panel.py`，4 个统计卡片 + 3 个柱状图 + 失败列表，30s 自动刷新）
- [x] P5-4 会话审计视图（`audit_panel.py`，按会话/事件类型筛选，16 种事件类型支持）
- [x] P5-5 发布规范（6 个 feature flag：task_mode/insight_index/memory_compress/risk_templates/evidence_capture/adb_tools，设置页 UI 开关 + 代码路径门控）

验收标准：

- 每次改动都有可量化前后对比
- 回归失败可快速定位
